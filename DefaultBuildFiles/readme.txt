This zip file is designed for deployment using the Microsoft Web Platform Installer.

http://www.microsoft.com/web/downloads/platform.aspx

We highly recommend using this to install Kartris - it makes it fast and easy to 
setup the software.

If you prefer to install manually however, just copy the "kartris" folder within 
this zip file to your webserver, and view the knowledgebase articles about installing
Kartris on our website

http://www.kartris.com
